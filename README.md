# Cybersecurity Risk Assessment & Mitigation Strategy
### E-Commerce Platform | Project Management Group Assignment

---

## Project Overview

This project focuses on identifying, analysing, and mitigating cybersecurity risks for an e-commerce platform. It includes a comprehensive project plan, risk management strategy, change request process, and version-controlled documentation using GitHub.

---

## Repository Contents

| File | Description |
|------|-------------|
| `README.md` | Project overview and repository guide |
| `risk-register.md` | Risk matrix, priority levels, and issue log |
| `change-log.md` | Change request form template and change records |
| `Project_Report.docx` | Full project report submitted for assessment |

## Project Details

- **Project:** Cybersecurity Risk Assessment and Mitigation Strategy for an E-Commerce Platform
- **Timeline:** 8 Weeks
- **Budget:** $50,000

---

## Branching Strategy

| Branch | Purpose |
|--------|---------|
| `main` | Final, reviewed documents only |
| `develop` | Working branch for ongoing changes |
| `risk-matrix-update` | Branch used to update the risk register |

All changes are made on a feature branch and merged into `main` via a pull request after peer review.
